Necesito un briefing para el desarrollo de la aplicación AstroBookings.
Se trata de un proyecto de ejemplo para usar en cursos y talleres de programación, sin intención de ser utilizada en producción.
La idea es ofrecer una solución a la empresa ficticia AstroBookings dedicada al turismo espacial.
Debe poder gestionar la flota de cohetes, planificar sus lanzamientos, anotar reservas sin superar la capacidad de cada cohete y suspender o cancelar los lanzamientos por cuestiones económicas o técnicas.
Para ello se plantea una API REST y una aplicación web para gestionar las reservas y los lanzamientos por parte de sus empleados.
